Rancang Bangun Sistem Pemantau Kursi Otomatis pada Kereta Api Berbasis IoT

by Irene tobingSept 8th, 2016

Rancang bangun sistem penumpang kereta api otomatis berbasis IoT terdapat sensor kelengkungan dan sensor suhu yang dipasang di tiap tempat duduk penumpang. Sensor ini berfungsi untuk mendeteksi apabila terdapat tempat duduk yang tidak terbeli tetapi terdapat penumpang yang menempati termpat duduk tersebut.

Perancangan sistem ini diharapkan dapat membantu petugas kereta api melakukan pemeriksaan melalui gerbong kendali secara realtime dan tidak diperlukannya lagi petugas berjalan ke tiap gerbong guna melakukan pemeriksaan pada penumpang, sehingga hal ini lebih efisien dibandingkan dengan pengecekan secara manual dan juga memakan waktu yang cukup singkat.
